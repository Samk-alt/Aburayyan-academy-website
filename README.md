# Abu-Rayyan Academy Website

## Overview
Official website for Abu-Rayyan Academy - A quality education institution in Mombasa, Kenya.

## Features
- 📱 Responsive Design
- 🎠 Image Carousel with auto-rotation
- 👨‍🏫 Teacher Directory
- 📍 Google Maps Integration
- 📞 Contact Information
- 💬 Social Media Links

## Sections
1. **Home** - Hero carousel with welcome messages
2. **Statistics** - School achievements and enrollment
3. **Programs** - School categories (Pre-Primary, Primary, Junior, Senior)
4. **About** - School mission and vision
5. **Teachers** - Interactive teacher directory
6. **Location** - Embedded Google Maps
7. **Contact** - Contact details and social links

## How to Use
1. Open `index.html` in a web browser
2. Navigate through sections using the navigation menu
3. Select a teacher from the dropdown to see their subject
4. Use carousel buttons or dots to navigate through hero images

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
- Google Maps Embed API
- Unsplash Images
